# Flask Notes App

A simple Notes CRUD Application built with Flask, Mysql and Bootstrap 5.

## Features

* Add Notes
* View Notes
* Edit Notes
* Delete Notes
* Responsive Bootstrap UI

## Technologies Used

* Python
* Flask
* Mysql
* Bootstrap 5

## Installation

```bash
git clone https://github.com/yourusername/flask-notes-app.git

cd flask-notes-app

pip install -r requirements.txt

python app.py
```

Open browser:

http://127.0.0.1:5000
